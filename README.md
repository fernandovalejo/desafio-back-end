# Desafio Back-End


### Requisitos:
* Ruby: `2.6.3`
* Rails: ` 6.0.3.4`

### Configuração 

```sh
$ bundle install
$ yarn install
$ rails db:create
$ rails db:migrate
$ rails db:seed
```

##### Ou use o script do projeto

```sh
$ ./init.sh
```

### Utilização 
Execute o comando abaixo dentro da pasta rails do projeto
```sh
$ rails s -b 0.0.0.0
```
